# 个人小站：一封写给你的信

这是一个纯静态的 GitHub Pages 页面，主入口是 `index.html`。  
页面风格是“书信感”，包含文字、图片、代码与数学公式的展示练习，并新增了一页信笺集。

## 访问方式

- 网站入口：https://huyaduo1.github.io/skills-communicate-using-markdown/
- 信笺集页面：`notes.html`

## 维护说明

- 主页文件是 `index.html`，请优先修改它。
- 样式与脚本拆分在 `css/style.css` 与 `js/main.js`。
- 如果 Pages 没有显示 `index.html`，请检查 GitHub 仓库的 `Settings` → `Pages`：
  - Branch 选择 `main`
  - Folder 选择 `/ (root)`
